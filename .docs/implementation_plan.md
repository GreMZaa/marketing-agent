# Implementation Plan: ИИ-Агент Маркетолог с Cohere API (MVP)

## 1. Стек технологий
* **Язык:** Python
* **UI-фреймворк:** Streamlit
* **LLM Интеграция:** Cohere API (библиотека `cohere`)
* **Данные:** Mock API (имитация ответов от рекламных площадок)

## 2. Основной функционал
Прототип агента-маркетолога должен выполнять следующие задачи:
* **Калькулятор метрик:** Агент получает сырые моковые данные (расход, показы, клики, покупки) и рассчитывает:
  * CPC = Расход / Клики
  * CPM = (Расход / Показы) * 1000
  * CPO = Расход / Покупки
  * CR = (Покупки / Клики) * 100
* **Генерация контента (Cohere API):**
  * На основе полученных метрик агент отправляет промпт с данными в Cohere API для генерации краткого текстового вывода (например, объяснение, почему кампания успешна или нет).
  * Генерация 2 новых вариантов рекламного текста для тех кампаний, которые показали плохие результаты.
* **Автоматическая оптимизация бюджета:**
  * Если CPO кампании выше заданного KPI -> агент снижает бюджет на 50% и предлагает новые креативы от Cohere.
  * Если CPO кампании ниже KPI -> агент увеличивает бюджет на 50% для масштабирования.

## 3. Структура файлов
1. `app.py` — основной файл Streamlit. Включает UI с полем для ввода API-ключа Cohere, кнопку "Запустить ИИ-агента" и дашборд с результатами расчетов и генераций.
2. `agent_logic.py` — классы для расчета метрик, управления бюджетами и взаимодействия с клиентом `cohere.Client(api_key)`.
3. `requirements.txt` — зависимости (`streamlit`, `pandas`, `cohere`, `python-dotenv`).

## 4. Сценарий демонстрации
Пользователь вставляет ключ Cohere API в интерфейс и нажимает кнопку запуска. Агент выводит на экран пересчитанные бюджеты, а также сгенерированные нейросетью Cohere пояснения и новые рекламные креативы.
